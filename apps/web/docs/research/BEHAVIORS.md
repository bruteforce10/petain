# Table22 Behaviors

- Sticky navigation stays at top with cream pill background and green text.
- Primary CTA and secondary CTA use rounded pill buttons with hover brightness/translate feedback.
- Hero uses horizontally scattered editorial cards; cards have soft shadows, rounded corners, and mixed rotations.
- Revenue section has tab labels: More revenue, Less lift, Deeper guest connections. Original is Webflow tabs; clone presents all value props visually.
- Industry section has category tabs: Restaurants, Wine Shops, Butchers & Fish Shops, Specialty Grocers, Bakeries, Retailers.
- Launch process is four sequential steps with content panels.
- Feature cards use hover lift and image-backed cards.
- Comparison section uses a three-column matrix with check and dash indicators.
- Responsive: desktop uses wide grids and image collages; mobile stacks sections, hides dense nav links, reduces heading scale.

Extracted colors:
- Deep green: rgb(0, 55, 46)
- Mid green: rgb(5, 87, 72)
- Cream: rgb(250, 250, 240)
- Warm grey: rgb(238, 238, 228)
- Black: rgb(0, 0, 0)

Fonts: computed Arial, sans-serif on key page elements.
